Indtroduction

This HTML code library is a responsive UI framework for iPortal 2, and documentation for installing the code so that a UI development environment can be maintained.

CSS and JS:

To have availability to the framework all pages must have the framework CSS and JS files included in the HTML pages. This can be achieved by copy-paste the following code into the HTML pages:

Copy-paste into the "" before all other stylesheets: 


                                <link rel="stylesheet" type="text/css" href="./css/mainStyles.css">
                                <link rel="stylesheet" type="text/css" href="./css/styleGuide-lib.css">
                                <link rel="stylesheet" type="text/css" href="./css/reset.css">
                                <link href="https://fonts.googleapis.com/css?family=Roboto:300,400,500,700" rel="stylesheet">
                                <link rel="stylesheet" type="text/css" href="./css/nc-icon.css">

Copy-paste into the HTML pages, near the tag. 


                                <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>
                                <!-- jQuery UI -->
                                <script src="https://code.jquery.com/jquery-1.12.4.js"></script>
                                <script src="https://ajax.googleapis.com/ajax/libs/jqueryui/1.12.1/jquery-ui.min.js"></script>
                                <!-- Latest compiled and minified Bootstrap JS -->
                                <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js" integrity="sha384-Tc5IQib027qvyjSMfHjOMaLkfuWVxZxUPnCJA7l2mCWNIpG9mGCD8wGNIcPD7Txa" crossorigin="anonymous"></script>

                                <!-- JS Plugins -->
                                <!-- Bootstrap Tour -->
                                <script src="js/plugins/bootstrapTour/bootstrap-tour.min.js"></script>
                                <!-- Kendo js plugins-->
                                <script src="https://kendo.cdn.telerik.com/2017.2.621/js/kendo.all.min.js"></script>
                                <!-- other plugins -->
                                <script src="js/plugins/metisMenu/jquery.metisMenu.js"></script>
                                <script src="js/plugins/slimscroll/jquery.slimscroll.min.js"></script>
                                <script src="js/plugins/masonary/masonry.pkgd.min.js"></script>
                                <!-- Bootstrap Tour -->
                                <script src="js/plugins/bootstrapTour/bootstrap-tour.min.js"></script>
                                <!-- custom jQuery script -->
                                <script src="js/custom.js"></script>

Each HTML UI component can be found on the specific page that corresponds to the UI component. When that code is used in conjunction with the specified CSS and JS code, the component will render correctly.
